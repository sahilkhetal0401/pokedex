# pokedex
An online Pokedex that fetches Pokémon data in real-time using the PokéAPI. Learn about Pokémon names, types, and images instantly.

# Pokedex V2
A simple interactive Pokedex built with HTML, CSS, and JavaScript.  
Search for any Pokémon and view its name, image, and type.  

## Features
- Search Pokémon in real-time
- Displays image, name, and types
- Mobile and desktop friendly
